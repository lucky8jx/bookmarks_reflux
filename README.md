# bookmarks_reflux

##拷贝这个例子

git clone git://github.com/lucky8jx/bookmarks_reflux.git

##安装依赖

npm install -g json-server

npm install

##开启模拟API服务器

json-server db.json

##正式开启本Demo

npm start
